# TraceKit Homebrew Tap

Official Homebrew tap for [TraceKit CLI](https://github.com/Tracekit-Dev/cli).

## Installation

```bash
brew install Tracekit-Dev/tap/tracekit
```

Or tap first, then install:

```bash
brew tap Tracekit-Dev/tap
brew install tracekit
```

## Upgrading

```bash
brew upgrade tracekit
```

## Documentation

For full documentation, visit the [TraceKit CLI repository](https://github.com/Tracekit-Dev/cli).

## About TraceKit

TraceKit CLI enables zero-friction APM setup for modern applications with:
- Single-command account creation
- Automatic framework detection
- SDK installation
- Health monitoring
- Distributed tracing

Get started in under 60 seconds with `tracekit init`.
